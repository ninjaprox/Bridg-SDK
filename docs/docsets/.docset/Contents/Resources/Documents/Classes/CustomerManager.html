CustomerManager

public class CustomerManager: NSObject, BridgSDKCentralDelegate

CustomerManager object is used to provide functionalities of customer.

CustomerManager is singleton class. Therefore, use sharedInstance property to obtain instance of this class. Before calling any methods, assure that setIdentity(_:) is called firstly to set customer’s identity, e.g., customer’s ID. Otherwise, an exception will be thrown.
