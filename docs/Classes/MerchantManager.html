MerchantManager

public class MerchantManager: NSObject, BridgSDKPeripheralDelegate

MerchantManager object is used to provide functionalities of merchant.

MerchantManager is singleton class. Therefore, use sharedInstance property to obtain instance of this class. Before calling any methods, assure to call setIdentity(_:) firstly to set customer’s identity, e.g., customer’s ID. Otherwise, an exception will be thrown.
